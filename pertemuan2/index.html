Sedang Latihan Favicon

Favicon HTML adalah sebuah tag HTML yang digunakan untuk menampilkan icon kecil yang muncul disebelah judul halaman web pada Browser,
Favicon ini biasanya digunakan untuk menampilkan logo atau icon situs web, sehingga memudahkan pengunjung untuk mengenali situs tersebut
